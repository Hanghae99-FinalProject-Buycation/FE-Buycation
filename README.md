
# 항해 10기 A반 6조의 [Buycation]

+ 함께 구매하고 나누고 즐기는 형태 공동 구매 플랫폼
+ 사이트 : https://buycation.co.kr/
+ 프로젝트 기간 2022.12.30 ~ 2022.02.10
+ 서비스 환경 : 웹 / 모바일
+ 사이트 목적
  1. 대량의 물품들을 근처 이웃들과 함께 구매하고 나누어 가격 부담을 덜어보세요!
  2. 배송비가 비싼 물품들을 함께 구매하거나 같은 음식점의 음식들을 함께 배달 시켜보세요!
  3. 모여서 함께 먹고 즐기는 건 어떨까요?
